# ngrinder
Install Ngrinder manually

### nGrinder controller, agent 실행
##### ngrinder 웹 페이지 접속
```
git clone https://github.com/anti1346/ngrinder.git
```
##### ngrinder 웹 페이지 접속
```
docker compose up -d
```
